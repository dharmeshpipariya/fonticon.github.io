Features

Designed for everyone, everywhere

FontIcon makes front-end web development faster and easier. It’s made for folks of all skill levels, devices of all shapes, and projects of all sizes.

Icon Generator

FontIcon ships with vanilla CSS, but its source code utilizes the two most popular CSS preprocessors, Less and Sass. Quickly get started with precompiled CSS.

Icon Packs

FontIcon easily and efficiently scales your websites and applications with a single code base, from phones to tablets to desktops with CSS media queries.

Cloud Syncing

With FontIcon, you get extensive and beautiful documentation for common HTML elements, dozens of custom HTML and CSS components, and awesome jQuery plugins.


Fork the project on GitHub

FontIcon is open source! It’s hosted, developed, and maintained on GitHub by folks like you.
